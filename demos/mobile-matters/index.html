How now mobile cow?

It can be done!

  • Keep it Quick

    Put important content at the top.

    Keep text blocks small.

    Compress images to make the site load faster.

    Use CSS3 instead of images.

  • Simplify Navigation

    Make it clear.

    Avoid rollovers.

    Have as few links as possible.

  • Be Thumb-Friendly

    Use large buttons.

    Increase clickable area.

    Use labels for clickable inputs.

  • Make it visibile

    Make sure it all fits on the screen without zooming.

    Make buttons obvious.

    Use CSS3 media queries to resize based on width. Go ahead, resize me.

  • Make it accessible

    Find alternatives to Flash.

    Use HTML5 and CSS3 for animation.

    Adapt for different orientations.

    CSS3 ADOBE EDGE

  • Make video accessible

    Use HTML5 video.

  • Make images scale

    Try to avoid loading unecessary images, but if you must - make sure they scale.

  • Aid Conversion

    Reduce steps on forms.
